# Авто Ру

## Предметная область: 
Покупка и продажа транспортных средств (Основной упор на автомобили)

## Решаемая проблема:
Упрощение и ускорение процесса купли/продажи транспортных средств.
- Для _продавца_ - снижение ресурсов, затраченных на рекламу и поиск клиентов, возможность легко охватить большую аудиторию
- Для _клиента_ - удобное сравнение вариантов, фильтрация по нужным параметрам и сведение риска обмана к минимуму

Также для обеих сторон это способ сократить время и усилия на коммуникацию и оформление сделки

## Сущности:
- Пользователь (userID, ФИО, email, номер телефона, дата регистрации)
- Продавец (sellerID, userID [FK], тип {частное лицо / дилер}, рейтинг, число объявлений)
- Объявление (adID, sellerID [FK], transportID [FK], заголовок, описание, стоимость, дата публикации, статус {активно, продано, заблокировано})
- Договор купли/продажи (contractID, sellerID [FK], userID [FK], transportID [FK], дата заключения, сумма, статус {подписан, в процессе})
- Транспортное средство (transportID, марка, модель, год выпуска, цвет, тип кузова, коробка передач, тип топлива, мощность, состояние, VIN)
- Автотека (autotekaID, transportID [FK], число владельцев, участие в ДТП, пробег, данные о страховках, использование в такси)
- Фото (photoID, adID [FK], url)
- Избранное (likeID, userID [FK], adID [FK])
- Отзыв (responseID, userID [FK], sellerID [FK], текст, рейтинг, дата публикации)
- Модератор (moderatorID, userID [FK], дата назначения, роль)
