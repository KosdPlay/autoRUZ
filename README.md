# Авто Ру

## Предметная область: 
Покупка и продажа транспортных средств (Основной упор на автомобили)

## Решаемая проблема:
Упрощение и ускорение процесса купли/продажи транспортных средств.
- Для _продавца_ - снижение ресурсов, затраченных на рекламу и поиск клиентов, возможность легко охватить большую аудиторию
- Для _клиента_ - удобное сравнение вариантов, фильтрация по нужным параметрам и сведение риска обмана к минимуму
Также для обеих сторон это способ сократить время и усилия на коммуникацию и оформление сделки
